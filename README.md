# DXC Framework CLI
A command line utility for the DXC Framework.

### Installation

#### From NPM
```bash
npm install -g dxcf-cli
```

#### From Source
```bash
mkdir -p ~/src/dxctechnology
cd ~/src/dxctechnology
git clone git@github.com:dxctechnology/dxcf-cli.git
cd dxcf-cli
npm install -g .
npm link
```

### Usage
```bash dxcf <command> [sub-command] [options]
```
